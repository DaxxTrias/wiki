**Vinetto** is a forensics tool to examine
[Thumbs.db](thumbs.db.md) files. It is a command line Python
script that works on [Linux](linux.md), [Mac OS
X](mac_os_x.md) and [Cygwin](Cygwin "wikilink") (win32).
