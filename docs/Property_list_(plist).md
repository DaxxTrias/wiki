The **Property List (plist)** format is widely used within [Mac OS
X](Mac_OS_X "wikilink"). It is used to store various kinds of
information, e.g. configuration.

There are multiple format variations of plists:

- ASCII plist
- Binary plist
- XML plist
- JSON plist
